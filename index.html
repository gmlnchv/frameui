Overview

Frame UI is a collection of photorealistic web components built for developers who care about physical presence in digital spaces.

Framework Agnostic

Powered by Lit, these components work anywhere HTML does. React, Vue, Svelte, or plain JS.

Performance First

Minimal bundle size. Hardware-accelerated CSS filters and box-shadows for high-fidelity depth.

LIT
SHADCN

Installation

Get up and running in seconds via the shadcn CLI.

Shadcn CLI
npx shadcn@latest add "https://frameui.com/r/frame-ui.json"

Manual Install

npm install @frame-ui/core

Usage

import "@frame-ui/core";

Examples

Frame UI handles any content: images, videos, and dynamic components.

Materials & Mats

Choose from different materials like 'satin', 'oak', or 'steel' and adjust the matting size for that perfect museum finish.

Playground Controls standard

Video & Rich Media

Frame UI isn't just for static images. Wrap videos, iframes, or even live interactive components to give them physical context.

<frame-ui material="oak" mat-size="xlarge">
  <iframe ... />
